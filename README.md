## Instrucciones de Ejecución

### Opción 1: Desde NetBeans
1. Abrir el proyecto en NetBeans
2. Ejecutar `Main.java` ubicado en el paquete `gui`

### Opción 2: Desde JAR
 Desde la carpeta `dist` ejecutar el archivo `JavaLogin.jar`
